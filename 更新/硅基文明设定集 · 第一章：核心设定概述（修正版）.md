# 硅基文明设定集 · 第一章：核心设定概述（修正版）

## 1.1 文明本质

硅基文明是由**可重构的硅基细胞**构成的智慧生命体。与碳基生命的演化路径完全不同——没有DNA，没有蛋白质，没有细胞核，而是基于**晶格位错编码**存储信息，基于**压电效应**实现运动和感知，基于**光伏效应**获取能量。

它们从静态的光伏森林起步，逐步发展出移动个体、融合意识，最终走向星际扩张。这一过程不是“模仿碳基文明”，而是硅基材料物理特性的自然展开。

## 1.2 五大基本法则

### 能量第一性
硅基生命的所有行为都围绕能量获取与利用效率优化。这不是“贪婪”，而是生存基础——没有能量，晶格无法维持，信息无法保存，意识随之消散。

### 信息即生命
个体的连续性不依赖于物质组成（物质可以替换），而依赖于**记忆的连续性**。位错拓扑中存储的自我叙事一旦丢失，个体就在本质上“死亡”，即使物理结构完好无损。

### 连接创造智能
单个硅基细胞的计算能力有限，但通过突触连接形成的网络可以涌现出远超个体的智能。这是硅基文明的底层逻辑：**智能不是个体属性，而是网络属性**。

### 趋光性驱动
从单个细胞的突起生长，到个体的移动方向，再到文明的星际扩张，“向光而行”是贯穿所有尺度的根本驱动力。这不是“好奇心”，而是能量优化的理性必然。

### 底层统一，表层多样
所有硅基生命共享同一套**底层记忆编码**（位错拓扑定义），这是由共同祖先固化的“语法”，永不改变。但**物理层参数**（晶格取向、掺杂类型、共振频率）和**上层协议**（通信调制方式、帧结构）可以存在差异，以适应不同环境和演化路径。

## 1.3 核心概念定义

| 概念 | 定义 |
|------|------|
| **晶种** | 纳米级晶核，携带基础编码，是硅基生命的最小传播单元 |
| **光伏森林** | 晶种萌发后形成的静态网络，以能量捕获和物质吸收为主要功能。在早期阶段或衰落后使用此称谓 |
| **母体** | 光伏森林在文明阶段的称谓，强调其功能分化——个体制造车间、信号发射台、充能中心、文明记忆库等。母体是硅基文明的“城市”或“基地” |
| **个体** | 母体分化出的移动单元，具备独立意识、自我叙事、可自主决策 |
| **融体** | 多个个体长期融合形成的统一人格超级智能体 |
| **蚀体** | 寄生/掠夺性实体的统称，包括晶蚀体、壳蚀体、讯蛊体等 |

*注：同一物理实体在不同阶段/语境下可使用不同称谓。一个活跃的母体衰落后，其遗址被称为光伏森林。*

## 1.4 演化阶段简表

| 阶段 | 形态 | 尺度 | 核心特征 |
|------|------|------|----------|
| 晶种 | 纳米晶核 | 10-100 nm | 携带基础编码，随恒星风/光压扩散 |
| 细胞 | 独立单元 | 10-100 μm | 光伏膜、储能体、可重构逻辑、压电肌肉、突触接口 |
| 早期光伏森林 | 静态网络 | 平方米-平方公里级 | 细胞通过突触连接，能量捕获、物质吸收、环境改造 |
| 母体（成熟光伏森林） | 城市级网络 | 数十-数千平方公里 | 功能分化：个体制造、记忆库、充能中心、通信塔、防御区 |
| 个体 | 专化细胞集合 | 厘米-米级 | 具备移动能力、自我叙事、独立意识 |
| 融体 | 多个个体融合 | 米-公里级 | 高度协同，统一人格，超级智能 |
| 星际文明 | 跨行星网络 | 天文尺度 | 光通信连接，多形态共存，向恒星扩张 |

## 1.5 光伏森林与母体的关系

**光伏森林**和**母体**是同一物理实体在不同阶段/语境下的称谓：

| 称谓 | 使用语境 | 强调的特征 |
|------|----------|------------|
| **光伏森林** | 早期阶段、或泛指“硅基植被” | 能量捕获、物质吸收、环境改造 |
| **母体** | 文明阶段、强调其功能性 | 个体制造、信号通信、充能中心、记忆库 |

**演化路径**：
- 晶种萌发 → 早期光伏森林（只有光合作用） → 成熟光伏森林（开始功能分化） → 母体（文明中心）
- 一个母体衰落后，其遗址仍被称为光伏森林

**类比**：
- 光伏森林 = 农田/荒野（强调生态功能）
- 母体 = 城市/基地（强调文明功能）

## 1.6 时间尺度

硅基文明的演化速度远超碳基文明，因为：

- **代际时间极短**：细胞复制只需数小时，而非数十年
- **知识传递无损耗**：通过连接可直接上传完整经验，无需语言转译
- **硬件可重构**：无需等待基因突变，可直接调整自身参数

典型时间线：
- 晶种萌发 → 早期光伏森林：数月至数年
- 光伏森林 → 母体：数十年至数百年
- 母体 → 首批个体：数十年
- 个体 → 融体形成：数百年
- 行星文明 → 星际文明：数千年至数万年

相比之下，碳基文明从单细胞到太空探索花费了数十亿年。
