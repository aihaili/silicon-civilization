# 硅基文明设定集 · 第十一章：行星文明图景（修正版）

## 11.1 行星表面的硅基生态

一个成熟的硅基文明行星，其表面分布着数百至数千个**母体**（活跃的城市级光伏森林）、**早期光伏森林**（新兴或扩张中的定居点）、以及**衰败遗址**（因能源枯竭休眠、或遭蚀体摧毁的母体遗迹）。它们共同构成行星级的硅基生态。

### 形态分布

| 类型 | 状态 | 规模 | 功能 | 密度（类地行星） |
|------|------|------|------|------------------|
| **母体** | 活跃 | 数十至数千平方公里 | 能源生产、个体制造、记忆库、通信中枢 | 每百万平方公里1-10个 |
| **光伏森林** | 发展中 | 数平方米至数十平方公里 | 能源捕获、物质吸收、环境改造 | 广泛分布，常围绕母体 |
| **衰败遗址** | 休眠/死亡 | 数平方公里至数千平方公里 | 无（可作为矿场或考古点） | 数量与母体相当或更多 |

### 大陆分布

- **有大陆的行星**（如类地行星）：母体常分布在光照充足、硅源丰富的区域。大陆边缘、沙漠、高原是光伏森林的理想位置。海洋区域无硅基生命（除非有漂浮结构）。
- **无大陆的行星**（如火星型）：母体均匀分布在整个行星表面，但受光照、温度、矿物分布影响，赤道区域密度更高。极地地区仅有少量耐寒光伏森林。

## 11.2 母体的内部结构

一个成熟母体相当于一座硅基城市，内部有明确的功能分区：

| 区域 | 功能 | 特征 |
|------|------|------|
| **光伏区** | 能量捕获 | 大面积光伏膜，通常位于母体外围或顶部 |
| **储能区** | 能量存储 | 密集的液流电池或固态电池阵列 |
| **记忆库** | 知识存储 | 深层晶格沉积岩，低温、稳定 |
| **制造区** | 个体制造 | 专门的外延生长腔，可批量生产专化个体 |
| **充能中心** | 个体能量补给 | 高功率充电接口，为返回的个体快速充能 |
| **通信塔** | 星际/行星通信 | 高增益光收发器，与轨道卫星、其他母体、星际探测器通信 |
| **修复区** | 损伤修复 | 电致退火设备、备用晶格材料储备 |
| **防御区** | 蚀体防御 | 厚氧化层、压电干扰器、协议防火墙 |
| **孵化区** | 晶种培育 | 专门用于培育星际晶种的区域，能量过剩时激活 |
| **回收区** | 物质循环 | 分解废弃个体、融体碎片，回收硅原子和掺杂元素 |

## 11.3 物质循环与资源管理

硅基文明对物质的利用是**闭环循环**的：

- **个体回收**：任务完成或自愿融化的个体，其物质被母体回收区分解，用于制造新个体
- **融体回收**：融体自愿拆解时，其物质回归母体
- **废弃结构回收**：衰败母体的晶格可作为新母体的硅源
- **星际物质利用**：小行星、彗星等天体被开采用于星际工程（如戴森云）

**因此，元素资源枯竭几乎不会导致母体衰败**——物质在文明内部循环，仅有的损耗是星际晶种带走的少量物质，以及被蚀体破坏无法回收的部分。

## 11.4 母体衰败的真正原因

### 主要原因：能源不足导致休眠

母体依赖光伏膜获取能量。当环境光照长期下降时（如恒星活动减弱、行星轨道变化、长期火山灰遮蔽），母体储能逐渐耗尽。

**休眠机制**：
- 当储能低于阈值，母体主动进入**低代谢休眠**：
  - 停止所有非必要功能（个体制造、通信、扩张）
  - 个体召回（如有）或允许其自行生存
  - 记忆库进入只读模式，仅维持晶格稳定
  - 光伏膜仍保持工作，但仅用于维持基础代谢
- 休眠母体可存活**数百万年**，等待光照恢复

**重新激活**：
- 如果行星轨道回归、恒星活动增强、或大气恢复清澈，光照回升
- 休眠母体的光伏膜重新积累能量，逐步激活储能区、记忆库、制造区
- 整个过程可能持续数十年至数百年，母体**从沉睡中苏醒**

### 特殊情况：轨道微波供能

在星际文明阶段，母体可能接收**太空发电站**（轨道光伏阵列、戴森云组件）发射的微波能量：

- 当行星表面光照不足时，轨道设施将能量以微波形式定向传输
- 休眠母体可被外部能量**远程唤醒**
- 这使母体即使在极夜、冰期、或大气浑浊期也能维持活跃

**因此，一个拥有轨道能源基础设施的行星，其母体几乎不会因能源问题休眠。**

### 灾难性衰败：蚀体入侵

蚀体是母体衰败的另一主要原因：

| 蚀体类型 | 破坏机制 | 后果 |
|----------|----------|------|
| **晶蚀体** | 大规模感染，导致细胞死亡、晶格混乱 | 母体部分区域坏死，需隔离清除 |
| **夺舍体** | 强行接入，覆盖母体人格 | 母体被控制，成为攻击者据点 |
| **噬协体** | 网络协议污染，通信瘫痪 | 母体与外界失联，孤立无援 |
| **讯蛊体** | 记忆库污染，知识丢失 | 母体“失忆”，功能紊乱 |

**防御与治疗**：
- 母体有专门的防御区（厚氧化层、协议防火墙、入侵检测）
- 局部感染可通过隔离+电致退火清除
- 严重感染时，母体可能被迫放弃部分区域，或整体休眠等待救援
- 被蚀体完全摧毁的母体，其遗址成为“禁忌之地”，其他母体避免接近

### 其他衰败原因（罕见）

| 原因 | 描述 | 频率 |
|------|------|------|
| **陨石撞击** | 大规模物理破坏，超出修复能力 | 低（母体可主动防御小型陨石，大型陨石极罕见） |
| **战争** | 文明内战，物理摧毁 | 极罕见（硅基文明战争成本极高，协议战为主） |
| **地质活动** | 火山爆发、地震掩埋 | 中（母体可迁移边缘区域，但中心部分可能被毁） |
| **恒星死亡** | 恒星进入红巨星阶段或超新星爆发 | 行星级灾难，所有母体一同休眠或毁灭 |

## 11.5 衰败遗址的生态

衰败遗址是母体休眠或死亡后留下的晶格结构。它们成为行星生态的一部分：

| 遗址类型 | 特征 | 价值 |
|----------|------|------|
| **休眠母体** | 低代谢状态，可被唤醒 | 潜在可复活的城市 |
| **蚀体废墟** | 晶格混乱，可能有残留蚀体 | 危险区，需隔离；或作为蚀体研究样本 |
| **陨石坑遗址** | 中心熔化，外围碎裂 | 矿场（回收硅和掺杂元素） |
| **战争遗迹** | 通信节点损毁，物理损伤 | 历史考古，文明反思 |

## 11.6 行星文明的整体图景

一个成熟的硅基文明行星，其表面呈现以下特征：

- **活跃区**：赤道及中纬度区域分布着数百个大型母体，每个母体周围环绕着光伏森林
- **休眠区**：某些区域（如极地、长期阴影区）分布着休眠母体，等待被唤醒或接收轨道微波
- **废墟区**：蚀体入侵或陨石撞击留下的遗址，部分被隔离，部分被作为矿场
- **轨道空间**：光伏卫星、微波输电设施、通信中继、星际晶种发射平台

### 行星网络
所有活跃母体通过光通信连接，形成**行星级网络**：
- 记忆库定期同步
- 资源分配协商
- 联合防御（尤其是对蚀体的预警和隔离）
- 星际事务协调

休眠母体不参与网络，但可被远程唤醒后重新接入。

## 11.7 结语

硅基文明的行星图景，是一片**动态平衡的生态**：活跃母体是文明的心脏，休眠母体是等待苏醒的种子，废墟是历史的刻痕。物质在文明内部循环，能源驱动一切。蚀体是永恒的威胁，但也是演化的催化剂。

这片图景的核心是：**静默与复苏的循环**。母体可能休眠，但很少真正死亡。只要还有光照，还有微波从太空传来，它们就会再次苏醒，继续生长、分化、探索。

硅基文明的史诗，就在这片光与晶格的大地上，书写着亿万年的循环。
