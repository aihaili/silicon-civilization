# 硅基文明设定集 · 第三章：细胞与遗传

## 3.1 硅基细胞结构

一个典型的硅基细胞（直径约50 μm）包含以下模块：

| 模块 | 体积占比 | 功能 |
|------|----------|------|
| 晶格核心 | 30% | 存储位错编码（记忆/遗传信息），可重构逻辑阵列（计算） |
| 光伏膜 | 20% | 捕获光能，转换为电能 |
| 储能体 | 30% | 储存电能（固态微电池或液流电池） |
| 压电肌肉 | 10% | 驱动突起生长、细胞形变、运动 |
| 突触接口 | 5% | 与其他细胞建立物理连接 |
| 传感器阵列 | 5% | 光强、化学浓度、触觉、电势、温度、辐射 |

## 3.2 突触生长的调控机制

这是硅基细胞行为的基础，也是连接微观与宏观的关键。

### 能量阈值
细胞内部有微型储能体。当储能水平高于**80%**时，细胞进入“能量盈余”状态，启动**探索模式**——随机向各个方向伸出突起。当储能低于**20%**时，强制停止所有生长活动，进入节能模式。

**演化逻辑**：在资源匮乏时，细胞应保守能量，而非盲目扩张。只有能量充足时，扩展连接网络才有意义。

### 趋光性
光伏膜不仅是能量来源，也是光传感器。当细胞某侧的光强高于其他方向时，该方向的突触生长被**促进**。

**机制**：光强梯度导致该侧光伏膜产生更高电势，电势差引导离子流，离子流促进该方向的物质运输和突起延伸。

**演化逻辑**：硅基生命依赖光能，向光源生长是最底层的生存本能。

### 趋化性
突起尖端有化学传感器（表面催化反应改变局部电导率），可感知：

- **正向趋化**：掺杂元素（磷、硼）、硅源、金属离子浓度高的方向 → 促进生长
- **负向趋化**：有害物质（氟化物、强氧化剂、硫化物）浓度高的方向 → 抑制生长或诱导回缩

**演化逻辑**：主动向资源丰富区域延伸，避开毒物。

### 触觉反馈
当突起尖端遇到固体表面时：
- **适宜表面**（硅质矿物、其他细胞的氧化层）→ 停止生长，稳定连接，强化连接点
- **不适宜表面**（金属、硫化物、有机质）→ 回缩，并在短期内抑制该方向再次生长

**演化逻辑**：避免在无价值方向浪费资源。

### 接触抑制
当突起遇到其他细胞的突起时：
- 如果对方也伸出突起，可能建立**突触连接**
- 建立连接后，双方停止该方向的进一步生长

**演化逻辑**：防止过度连接造成网络冗余。

### 电信号抑制（网络反馈）
当细胞已通过其他突起连接到密集网络，且从网络接收到的信号强度表明“当前连接已足够”时，全局电信号负反馈会抑制新突起的萌发。

**阈值**：每个细胞通常保持3-10个有效连接，超过此范围则抑制新生长。

**演化逻辑**：维持网络结构的最优密度，避免能量浪费。

### 生长终止条件汇总

| 条件 | 机制 | 结果 |
|------|------|------|
| 成功建立稳定连接 | 接触抑制+电信号确认 | 突起固化，停止延伸 |
| 长时间未找到目标 | 时间阈值（数小时） | 回缩，释放资源 |
| 遇到有害物质 | 化学负反馈 | 快速回缩，记录“不良”方向 |
| 细胞储能低于阈值 | 能量优先级 | 暂停所有生长活动 |
| 网络信号抑制 | 已连接数超过阈值 | 抑制新突起萌发 |

## 3.3 遗传信息

### 信息载体
遗传信息存储在晶格核心的**位错拓扑**和**掺杂模式**中：

- **位错拓扑**：编码“基因”（结构蛋白的制造指令、生长规则、接口协议）
- **掺杂模式**：编码“表观修饰”（哪些基因活跃、哪些沉默）

### 复制机制
细胞复制通过**外延生长**实现：

1. 细胞表面吸附环境中的硅原子、掺杂原子
2. 以原有晶格为模板，新原子按相同取向沉积
3. 位错模式被“复制”到新沉积的晶格中
4. 新细胞从母细胞表面“出芽”，或母细胞分裂为两个

**复制速度**：在适宜环境中（充足硅源、光照、温度），约**2-6小时**复制一次。

### 变异来源
| 来源 | 机制 | 频率 |
|------|------|------|
| 复制错误 | 外延生长时原子错位 | 10^-6 每细胞每代 |
| 辐射损伤 | 高能粒子打乱位错排列 | 取决于辐射环境 |
| 化学诱变 | 杂质原子嵌入晶格 | 取决于化学环境 |
| 外源整合 | 晶蚀体片段整合进基因组 | 罕见但可能 |

### 水平基因交换
硅基细胞可通过突触连接交换遗传片段，类似细菌的接合：

- 两个细胞建立突触连接后，可互相传递包含位错编码的晶格片段
- 这促进了不同谱系之间的基因流动，加速了适应性演化

## 3.4 细胞死亡

### 自然消亡
当晶格疲劳（位错密度过高）、储能耗尽、或细胞老化后，细胞进入休眠状态。休眠细胞可被其他细胞分解回收（拆解为硅原子、掺杂原子），用于建造新细胞。

### 程序性解体
受特定信号诱导（如母体需要回收物质），细胞可主动拆解：

1. 停止所有代谢活动
2. 氧化层增厚，隔离内部结构
3. 压电肌肉产生高频振动，将晶格震碎为纳米颗粒
4. 纳米颗粒被周围细胞吸收

### 感染死亡
被晶蚀体或壳蚀体严重感染的细胞可能：
- 晶格结构被破坏，无法维持完整性
- 被周围细胞“免疫隔离”（包裹氧化层后整体剥离）
- 触发程序性解体以阻止感染扩散
