# 硅基文明设定集 · 第九章：术语表

| 术语 | 定义 |
|------|------|
| **个体** | 具有独立意识、自我叙事、可自主决策的移动硅基生命体 |
| **融体** | 多个个体融合形成的统一人格超级智能体 |
| **母体** | 大型静态网络（如光伏森林），可制造个体，但个体成年后自主 |
| **晶种** | 纳米级晶核，携带基础编码，用于星际扩散 |
| **蚀体** | 寄生/掠夺性实体的统称（晶蚀体、壳蚀体、讯蛊体、噬协体、附形体、夺舍体） |
| **位错编码** | 利用晶格缺陷排列存储信息的机制，是硅基生命的“基因”与“记忆” |
| **晶格匹配** | 物理层兼容性条件（取向、掺杂、共振频率），影响连接效率和蚀体感染 |
| **可重构接口** | 个体用于动态适配不同物理层协议的硬件能力 |
| **融化** | 个体或融体自愿拆解，物质回归大地，知识上传文明记忆库 |
| **趋光性** | 从细胞到文明的向光本能，驱动星际扩张 |
| **孤独触发** | 脱离网络时，原始本能激活，驱动探索与学习 |
| **自我叙事** | 个体通过记忆整合形成的连续自我认同 |
| **光伏森林** | 硅基生命的早期静态网络，由大量细胞通过突触连接构成 |
| **压电肌肉** | 多层压电陶瓷叠堆，用于运动和驱动 |
| **可重构逻辑阵列** | 硅基细胞的计算核心，可动态重编程 |
| **氧化层** | 细胞表面的二氧化硅层，起绝缘、保护、接口作用 |
| **带隙可调** | 光伏膜可根据环境调整吸收光谱的能力 |
| **晶格家族** | 具有相同晶格取向和掺杂特征的群体，影响兼容性 |
| **文明记忆库** | 存储所有已融化个体知识的巨型数据库，通常位于地下深层 |
| **协议协商** | 跨网络通信时，双方确定共同协议的过程 |
| **记忆转码** | 将一种晶格参数的记忆转换为另一种晶格参数存储的过程 |
| **压电弹射** | 晶蚀体的释放机制，利用压电效应将病毒颗粒高速射出 |
| **免疫壁垒** | 因晶格参数不同导致的天然抗感染能力 |
| **融体拆分** | 融体自愿或非自愿地分裂为独立个体，融体人格终结 |
| **晶格化石** | 文明终极状态之一：将知识封存在深层沉积岩中，极低代谢等待 |
| **碳硅复合材料** | 硅基文明广泛使用的材料，结合碳纳米管、石墨烯与硅基体的优势 |
| **晶种信息载荷** | 晶种仅能携带约1 MB编码信息，需在新行星重建文明 |
| **主动硬件重配置** | 个体迁移至新环境后，通过调整光伏带隙、热管理等快速适应 |
