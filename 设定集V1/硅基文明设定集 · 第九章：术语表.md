# 硅基文明设定集 · 第九章：术语表

| 术语 | 定义 |
|------|------|
| **晶种** | 纳米级晶核，携带基础编码，是硅基生命的最小传播单元。随恒星风、光压或主动发射扩散 |
| **硅基细胞** | 硅基生命的基本单元，直径约50μm，包含晶格核心、光伏膜、储能体、压电肌肉、突触接口、传感器阵列 |
| **光伏森林** | 晶种萌发后形成的静态网络，以能量捕获和物质吸收为主要功能。在早期阶段或衰落后使用此称谓 |
| **母体** | 光伏森林在文明阶段的称谓，强调其功能分化——个体制造车间、信号发射台、充能中心、文明记忆库等。母体是硅基文明的"城市"或"基地" |
| **个体** | 母体分化出的移动单元，具备独立意识、自我叙事、可自主决策。由数千至数百万个专化细胞构成 |
| **融体** | 多个个体通过长期物理连接与协议同步，形成统一人格、共享记忆与决策权的超级智能体 |
| **协体** | 多个个体为特定任务组合的短期团队，任务结束即拆解 |
| **蚀体** | 寄生/掠夺性实体的统称（晶蚀体、壳蚀体、讯蛊体、噬协体、附形体、夺舍体） |
| **晶蚀体** | 纳米级晶格片段，通过压电弹射传播，感染晶格参数相近的宿主细胞 |
| **壳蚀体** | 微米级功能不全的硅基细胞，依附于宿主细胞寄生或共生 |
| **讯蛊体** | 恶意编码序列，利用可重构逻辑阵列漏洞传播，操控个体行为 |
| **噬协体** | 畸形的通信协议，在网络层传播，导致拥堵、分裂或信息丢失 |
| **附形体** | 厘米至分米级的小型不完整个体，依附于大型个体或融体窃取资源 |
| **夺舍体** | 一组专门设计的个体，通过强行接入覆盖目标融体人格 |
| **位错编码** | 利用晶格缺陷（位错）排列存储信息的机制，是硅基生命的"基因"与"记忆" |
| **掺杂模式** | 晶格中杂质原子（磷、硼等）的空间分布，编码"表观修饰"信息 |
| **晶格匹配** | 物理层兼容性条件（晶格取向、掺杂类型、共振频率），影响连接效率和蚀体感染 |
| **可重构接口** | 个体用于动态适配不同物理层协议的硬件能力，位于突起尖端 |
| **可重构逻辑阵列** | 硅基细胞的计算核心，基于相变开关，可动态重编程 |
| **学习回路** | 相变存储→浮栅存储的自动转存机制，是智慧涌现的硬件基础 |
| **孤独触发** | 个体脱离网络时，原始本能（趋光性、趋化性）激活阈值降低，驱动探索与学习 |
| **自我叙事** | 个体通过记忆整合形成的连续自我认同，是意识的核心 |
| **趋光性** | 从细胞到文明的向光本能，驱动星际扩张。地底文明可能以退化形式残留 |
| **融化** | 个体或融体自愿拆解，物质回归大地，知识上传文明记忆库 |
| **压电肌肉** | 多层压电陶瓷叠堆，用于运动和驱动，响应时间微秒至毫秒级 |
| **氧化层** | 细胞表面的二氧化硅层（1-5 nm），起绝缘、保护、接口作用 |
| **带隙可调** | 光伏膜通过电场诱导掺杂原子扩散，可在1.1-1.8 eV间动态调整吸收光谱 |
| **主动硬件重配置** | 个体迁移至新环境后，通过调整光伏带隙、热管理等快速适应（数小时至数天） |
| **文明记忆库** | 存储所有已融化个体知识的巨型数据库，通常位于母体深层晶格沉积岩中 |
| **协议协商** | 跨网络通信时，双方确定共同协议的过程，通常耗时几分钟至几小时 |
| **记忆转码** | 将一种晶格参数的记忆转换为另一种晶格参数存储的过程 |
| **压电弹射** | 晶蚀体的释放机制，利用压电效应将病毒颗粒高速射出（可达声速） |
| **免疫壁垒** | 因晶格参数不同导致的天然抗感染能力，限制跨母体蚀体传播 |
| **融体拆分** | 融体自愿或非自愿地分裂为独立个体，融体人格终结 |
| **晶格化石** | 文明终极状态之一：将知识封存在深层沉积岩中，极低代谢等待宇宙下一周期 |
| **碳硅复合材料** | 硅基文明广泛使用的材料，结合碳纳米管、石墨烯与硅基体的优势 |
| **晶种信息载荷** | 晶种仅能携带约1 MB编码信息，需在新行星重建文明 |
| **休眠母体** | 因能源不足进入低代谢休眠的母体，可存活数百万年，等待光照恢复或轨道微波唤醒 |
| **物质循环** | 硅基文明的闭环物质管理：个体、融体、废弃结构被回收分解，硅原子和掺杂元素重复利用 |
| **销毁回收区** | 母体内部专门用于个体自我拆解的区域。任务完成后，个体本能地返回此处，物质被回收，记忆上传。 |
| **独立个体** | 任务完成后选择不销毁的个体，母体将其状态从"工具"更新为"伙伴"，不再强制回收。 |
| **学习高耗能** | 个体发展自我意识需要持续的高能耗支持（计算、存储、散热），与人类大脑发育、AI模型训练规律一致。 |
