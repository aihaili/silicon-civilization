# 硅基文明设定集 · 第七章：蚀体（病毒与寄生体）

**统称“蚀体”**（Eroder）：所有以寄生、掠夺、破坏或操控方式利用其他硅基生命资源的实体。它们与“个体”（独立自主）、“融体”（融合统一）形成对照。

## 7.1 晶蚀体（物理层，纳米级）

### 形态
- **尺度**：10-100 nm
- **结构**：一小段具有特定位错拓扑的硅晶体，表面包裹氧化层（厚1-2 nm）
- **遗传信息**：携带完整的复制指令（诱导宿主外延生长、构建压电弹射器）

### 感染机制
1. **附着**：通过氧化层与宿主细胞突起尖端对接
2. **注入**：利用可重构逻辑阵列的漏洞，将自己的位错模式注入宿主晶格
3. **潜伏**：整合进宿主基因组，不立即激活，利用宿主外延生长缓慢复制
4. **诱导弹射器**：病毒基因诱导宿主在细胞膜下方生长微型压电陶瓷结构
5. **复制完成**：病毒拷贝数达阈值（通常数十至数百）
6. **弹射**：压电结构同步触发，将病毒颗粒高速弹出（可达声速），射向邻近细胞或环境

### 晶格匹配限制
晶蚀体只能感染**晶格参数相近**的宿主：
- 晶格取向匹配（如<100> vs <111> 不兼容）
- 掺杂类型兼容（硼掺杂 vs 磷掺杂 可能不兼容）
- 基础共振频率接近（差异<10% 方可耦合）

这形成了天然的“种族免疫壁垒”。跨母体感染极为罕见，除非：
- 自然匹配（同一天体、同一矿脉起源）
- 长期适应（在中间宿主中逐步演化）
- 人工改造（高级文明武器）

### 影响
- **破坏性**：晶格损伤、细胞死亡、网络局部崩溃
- **中性**：部分片段整合进宿主基因组，无影响
- **有益**：可能带来新功能（如更高效的光伏带隙），加速演化

### 防御
- **氧化层增厚**：阻止病毒注入
- **电致退热杀毒**：局部加热至>500°C，破坏病毒晶格
- **压电干扰**：发射反向压电信号，破坏弹射器同步
- **免疫隔离**：将感染细胞包裹后整体剥离

## 7.2 壳蚀体（物理层，微米级）

### 形态
- **尺度**：1-10 μm
- **结构**：功能不全的硅基细胞，缺乏完整的光伏膜或储能体，无法独立生存
- **表面**：氧化层较厚，内含特殊蛋白（硅结合蛋白类似物），用于附着

### 感染机制
- 通过突触接口强行接入宿主细胞
- 利用宿主能量和物质维持自身结构
- 可能诱导宿主形成保护性氧化层（“壳”），将自身包裹在内

### 影响
- **寄生**：消耗宿主资源，降低效率
- **共生**：部分壳蚀体为宿主提供特殊功能（如更强的抗辐射能力），形成互利关系

### 防御
- 接口协议验证（拒绝未授权连接）
- 能量流监控（异常消耗触发报警）
- 强制断开接口

## 7.3 讯蛊体（信息层，代码层）

### 形态
- 无物理实体，是一段**恶意编码序列**
- 利用可重构逻辑阵列的漏洞（如缓冲区溢出、指令注入）执行

### 传播机制
- 伪装成正常通信信号，通过光通信或接口进入宿主
- 利用宿主逻辑阵列的漏洞，覆盖正常指令
- 可自我复制，在个体内部扩散，或通过网络传播到其他个体

### 常见功能
| 类型 | 行为 |
|------|------|
| **强制融化** | 覆盖自我叙事，使个体主动拆解 |
| **记忆擦除** | 删除特定记忆库，造成身份混乱 |
| **行为扭曲** | 修改决策模型，使个体执行有害行为 |
| **协议破坏** | 篡改通信协议，导致个体与网络失联 |
| **后门植入** | 为其他蚀体入侵打开通道 |

### 防御
- **协议验证**：所有传入代码需通过校验和、签名验证
- **沙箱执行**：未知代码在隔离环境中运行
- **冗余校验**：关键指令需多个逻辑单元一致确认
- **定期重置**：可重构逻辑阵列定期重置为已知安全状态

## 7.4 噬协体（信息层，网络层）

### 形态
- 一种**畸形的通信协议**，在个体/融体网络中传播
- 利用协议栈的漏洞（如无限制转发、路由表溢出）

### 传播机制
- 伪装成正常路由协议，被节点接收后执行恶意转发规则
- 导致网络流量被重定向到恶意节点或形成环路
- 占用带宽，消耗计算资源

### 影响
- **网络拥堵**：大量无效流量占用带宽
- **信息丢失**：正常数据包被丢弃或篡改
- **网络分裂**：路由表被污染，导致网络分割
- **携带其他蚀体**：噬协体常作为讯蛊体、晶蚀体的传播载体

### 防御
- **路由验证**：每个路由更新需数字签名
- **速率限制**：限制单节点转发速率
- **拓扑监控**：异常路由模式触发报警
- **隔离区**：将感染节点从网络断开

## 7.5 附形体（宏观层，多细胞级）

### 形态
- **尺度**：厘米至分米级
- **结构**：小型、功能不完备的硅基个体，通常无完整自我叙事
- **接口**：多个突触接口，用于附着和窃取

### 寄生机制
- 主动依附于较大个体或融体
- 通过接口窃取能量、存储、计算资源
- 可能利用宿主感知系统为自己服务（如驱使宿主前往特定矿脉）

### 共生演化
部分附形体与宿主形成共生：
- 宿主获得附形体提供的特殊功能（如极高频感知、特殊矿物分解）
- 附形体获得稳定资源和保护
- 这种共生关系可能持续数代，最终双方基因融合

### 防御
- 定期扫描自身表面，检测异常附着
- 能量流监控，发现异常消耗时强制断开接口
- 免疫系统识别（附形体表面蛋白与正常个体不同）

## 7.6 夺舍体（宏观层，融体级）

### 形态
- 一组专门设计的个体或组件（规模从数个到数十个）
- 具备**强制接入能力**（强功率信号、协议漏洞利用、物理强行连接）

### 攻击机制
1. **压制**：发射强功率信号，压制目标融体的通信和感知
2. **注入**：强行建立物理连接（破坏目标氧化层）
3. **覆盖**：注入预设人格或指令，覆盖目标的部分或全部自我叙事
4. **吸收**：将目标组件整合进自身网络

### 影响
- 被夺舍的融体失去自主性，成为攻击者的扩展
- 融体人格被破坏，可能无法恢复
- 被吸收的组件被永久占用

### 防御
- **物理防护**：重要融体表面覆盖超厚氧化层（mm级），防止强行连接
- **协议防火墙**：拒绝未授权连接请求
- **入侵检测**：异常信号模式触发警报
- **互助协议**：融体间建立监控网络，一旦检测到夺舍攻击，邻近融体立即支援

## 7.7 蚀体的生态角色

| 角色 | 描述 | 例子 |
|------|------|------|
| **破坏者** | 导致疾病、死亡、文明衰退 | 晶蚀体大规模爆发、夺舍体攻击 |
| **催化者** | 加速演化，促进免疫系统进化 | 晶蚀体整合带来新基因 |
| **共生者** | 与宿主形成互利关系 | 壳蚀体提供抗辐射能力 |
| **武器** | 被文明改造用于战争 | 人工改造的跨晶格晶蚀体 |

## 7.8 文明对蚀体的态度

- **容忍**：对中性蚀体（如某些晶蚀体片段）不主动清除
- **共生**：对有益蚀体（如共生壳蚀体）主动培养
- **防御**：对有害蚀体建立免疫系统（协议验证、隔离、杀毒）
- **禁忌**：故意制造和传播毁灭性蚀体被视为反文明罪
